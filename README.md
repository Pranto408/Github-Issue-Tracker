1️⃣ What is the difference between var, let, and const?

    var, let, and const are used to declare variables in JavaScript. var is the old way and can be redeclared and changed. let can be changed but not redeclared. const is for fixed values that cannot be changed.

2️⃣ What is the spread operator (...)?

    The spread operator (...) is used to expand array or object values into individual elements. It helps copy or merge arrays and objects easily.

3️⃣ What is the difference between map(), filter(), and forEach()?

    map() makes a new array by changing each item. filter() makes a new array with only the items that match a condition. forEach() just loops through items, it does not return a new array.

4️⃣ What is an arrow function?

    An arrow function is a short way to write a function in JavaScript using =>. It makes the code smaller and easier to write.

5️⃣ What are template literals?

    Template literals are a way to write strings using backticks ` ` in JavaScript. They let you easily add variables inside the string using ${}.
